# Cross-Project Synthesis

Synthesized pattern records are the preferred recommendation source once multiple projects independently solve the same problem.

A synthesis record should compare observed implementations, identify common elements, explain differing assumptions, summarize implementation complexity, and define the minimum useful extraction for ProjectForge-style local-first projects.

Synthesis records should link back to pattern cards, component cards, project cards, contradiction records, and adoption outcomes. They should not replace evidence files; they are compact decision-support summaries.
