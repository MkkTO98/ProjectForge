# Skill: State Update

State files are the active project memory.

Update:
- `state/recent_changes.md` after meaningful changes.
- `state/known_issues.md` when discovering unresolved problems.
- `state/lessons.md` after reusable lessons.
- `state/active_goal.md` when the current objective changes.

Do not put durable decisions only in state. Put them in `artifacts/decisions/`.
