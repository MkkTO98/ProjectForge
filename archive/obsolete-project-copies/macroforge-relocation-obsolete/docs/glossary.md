# Glossary

- Raw evidence: Immutable source payload or snapshot plus checksum and source metadata.
- Staging: Source-shaped normalized data loaded before canonical curation.
- Curated: Canonical dimensions and facts used for analysis.
- Lineage: Records connecting raw artifacts, transformations, database rows, and quality checks.
- Vintage/as-of date: The observation version/release context used to avoid overwriting historical revisions.
- Attribute set: Structured qualifiers that preserve source-specific dimensions without widening fact tables prematurely.
- Vertical slice: One real end-to-end pipeline proving extraction, storage, validation, and reporting.
