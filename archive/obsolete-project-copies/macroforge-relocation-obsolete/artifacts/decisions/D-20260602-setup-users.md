## Question
Who uses this project: only you, agents, public users, clients, or another group?

## Answer
Primary user is the owner; secondary users are constrained local agents acting as operator, planner, builder, reviewer, researcher/analyst, data steward, auditor, and teacher/explainer. No public users in v1.
