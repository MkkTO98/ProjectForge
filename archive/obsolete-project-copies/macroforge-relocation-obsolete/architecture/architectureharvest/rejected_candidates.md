# ArchitectureHarvest Rejected Candidates

Status: active

No ArchitectureHarvest recommendation is permanently rejected by this retrofit.

## Explicit near-term non-adoptions

- Broad dbt runtime adoption is not adopted.
- Broad Dagster runtime adoption is not adopted.
- Generalized ingestion framework extraction is not adopted.
- Automatic orchestration, scheduling, materialization, or live database writes are not adopted.

These are near-term boundaries, not permanent architectural prohibitions.
